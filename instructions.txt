Requisitos:
Eclipse instalado
Java 1.8
Weka

Steps:
1- open weka directory
2- create a temporary folder in your desktops
3- copy weka-src.jar contents to the temporary folder
4- open eclipse
5- import the new folder as a maven project
6- copy the classifier source to the maven project
7- run the project using the class weka.Run
8-  open weka directory
9- open weka.jar as archive
10- copy he contencts of build/classes/weka to weka/ on weka.jar
11- if you fail to add the files you probably dont have admin priviledges to change the files on C:/, if thats the case copy the weka folder to the desktop
